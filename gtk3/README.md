# PCM — Python Connection Manager (GTK3)

> La documentazione completa si trova nel **[README principale](../README.md)**.

Questa è la versione GTK3, quella in sviluppo attivo.  
Screenshot, istruzioni di installazione, elenco funzionalità e confronto con altri tool sono nel README della root del repository.

---

**Avvio rapido:**

```bash
python3 PCM.py
```

oppure, dalla root del repository:

```bash
bash setup.sh          # installa le dipendenze
cd gtk3 && python3 PCM.py
```
